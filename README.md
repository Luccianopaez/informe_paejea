# Informe Jurídico y Técnico — Caso Equifax 2017

Aplicación web desarrollada en React + Vite para el análisis jurídico y técnico del caso de ciberseguridad Equifax 2017, utilizando legislación chilena como base principal de evaluación.

---

# Descripción del Proyecto

Este proyecto consiste en una aplicación web académica desarrollada para analizar uno de los incidentes de ciberseguridad más importantes a nivel mundial: la filtración masiva de datos sufrida por Equifax en 2017.

El análisis se centra en responder la siguiente hipótesis:

> “Si el incidente Equifax hubiese ocurrido en Chile, ¿qué normas jurídicas chilenas aplicarían?”

La aplicación presenta el contenido del informe mediante archivos Markdown renderizados dinámicamente dentro de una interfaz moderna desarrollada con React.

# Contenido del Informe

La aplicación se encuentra organizada en 8 archivos Markdown:

| Archivo | Contenido |
|---|---|
| 01_resumen_jepae.md | Resumen ejecutivo del caso |
| 02_marco_jepae.md | Marco normativo nacional e internacional |
| 03_delitos_jepae.md | Tipificación de delitos informáticos |
| 04_comparacion_jepae.md | Comparación de marcos regulatorios |
| 05_responsabilidades_jepae.md | Responsabilidades legales |
| 06_datos_jepae.md | Tratamiento de datos personales |
| 07_conclusiones_jepae.md | Conclusiones y recomendaciones |
| 08_prompts_jepae.md | Bitácora de uso de IA |

---

# Objetivos del Proyecto

- Analizar un caso real de ciberseguridad.
- Aplicar legislación chilena sobre delitos informáticos.
- Evaluar responsabilidades jurídicas derivadas de filtraciones de datos.
- Analizar tratamiento de datos personales según Ley 19.628.
- Comparar marcos regulatorios internacionales.
- Desarrollar una aplicación web moderna para presentar documentación técnica y jurídica.

---

# Legislación Analizada

## Nacional

- Ley N.º 21.459 sobre delitos informáticos
- Ley N.º 19.628 sobre protección de datos personales
- Constitución Política de la República de Chile

## Internacional

- GDPR (Unión Europea)
- Convenio de Budapest
- Computer Fraud and Abuse Act (Estados Unidos)
