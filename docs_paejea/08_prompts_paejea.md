## Investigación de Casos de Ciberseguridad

Busca casos reales de ciberseguridad que hayan tenido impacto mundial y que permitan desarrollar un análisis jurídico basado en legislación chilena.

Los casos deben incluir elementos como:
- filtración masiva de datos,
- ransomware,
- delitos informáticos,
- protección de datos personales,
- responsabilidad empresarial,
- y comparación de marcos regulatorios internacionales.

Dame solamente 5 casos reconocidos internacionalmente para poder elegir el más adecuado para un informe académico desarrollado en React y Markdown.

## Selección del Caso Equifax

Necesito un caso que tenga suficiente información pública, relevancia jurídica y relación con protección de datos personales para poder aplicar:
- Ley 21.459 de Chile,
- Ley 19.628,
- derechos ARCO,
- y comparación con normativas internacionales como GDPR.

Recomiéndame cuál de los casos es el más completo para obtener un buen análisis académico.
