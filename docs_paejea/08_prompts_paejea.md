# Prompts Utilizados

## Investigación de Casos de Ciberseguridad

Busca casos reales de ciberseguridad que hayan tenido impacto mundial y que permitan desarrollar un análisis jurídico basado en legislación chilena.

Los casos deben incluir elementos como:
- filtración masiva de datos,
- ransomware,
- delitos informáticos,
- protección de datos personales,
- responsabilidad empresarial,
- y comparación de marcos regulatorios internacionales.

Dame solamente 5 casos reconocidos internacionalmente para poder elegir el más adecuado para un informe académico desarrollado en React y Markdown.

## Selección del Caso Equifax

Necesito un caso que tenga suficiente información pública, relevancia jurídica y relación con protección de datos personales para poder aplicar:
- Ley 21.459 de Chile,
- Ley 19.628,
- derechos ARCO,
- y comparación con normativas internacionales como GDPR.

Recomiéndame cuál de los casos es el más completo para obtener un buen análisis académico.

## Organización de la Documentación Markdown

Ayúdame a estructurar un proyecto académico donde el contenido principal estará separado en 8 archivos Markdown.

Cada archivo debe representar una sección independiente del informe:
- resumen ejecutivo,
- marco normativo,
- delitos informáticos,
- comparación regulatoria,
- responsabilidades,
- tratamiento de datos,
- conclusiones,
- y prompts utilizados.

La estructura debe verse profesional y pensada para integrarse posteriormente en una aplicación React.

## Configuración de React con Vite

Explícame cómo crear un proyecto moderno utilizando React con Vite desde cero.

Quiero una estructura limpia, rápida y profesional para posteriormente desplegar el proyecto en Vercel.

Incluye:
- creación del proyecto,
- instalación de dependencias,
- estructura recomendada de carpetas,
- y buenas prácticas iniciales.

## Instalación de Tailwind CSS

Ayúdame a instalar Tailwind CSS en un proyecto React creado con Vite.

Explícame:
- qué dependencias instalar,
- cómo configurar el archivo CSS principal,
- cómo habilitar las clases utility,
- y cómo verificar que Tailwind funciona correctamente dentro del proyecto.

Quiero seguir buenas prácticas y dejar la configuración preparada para una interfaz moderna y responsive.

## Uso de Lucide Icons

Explícame cómo instalar y utilizar Lucide Icons dentro de un proyecto React con Vite.

Necesito:
- instalación correcta del paquete,
- importación de iconos,
- ejemplos de uso en componentes,
- personalización de tamaño y estilos,
- y recomendaciones para mantener consistencia visual dentro de la interfaz.
